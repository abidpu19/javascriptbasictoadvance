Array Function - Built in

The Array.forEach() function takes another function as parameter and invokes the function for every item in the array. Also while invoking the function, it also will pass the item as a parameter to the callback function. Array.forEach(callback); // callback is the function that gets invoked for each item of the array

map() Function

Just Like forEach(), map() also iterates over each element in the array and invokes a function. But the difference is, forEach() just invokes function on each item in the array. filter()

What if we want to get all words whose length is greater than 5? For this we can use filter(). filter also takes callback function as a parameter. It iterates over each element and creates a sub array containing the elements that satisfy the condition given in the callback. Here is a simple example demonstrating the implementation of filter() function

find()

Array has a method called find(). It returns the first element in the array which satisfies a given condition. It takes a callback. It executes the callback for each element in the array. If the callback returns true, then find returns the element for which the callback returned true and stops further iteration. If it was false for all elements, it returns undefined.
